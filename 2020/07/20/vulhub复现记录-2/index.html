Joomla

Joomla!是一套全球知名的内容管理系统，是使用PHP语言加上MySQL数据库所开发的软件系统

Joomla 3.4.5 反序列化漏洞（CVE-2015-8562）

影响版本
Joomla 1.5.x, 2.x, and 3.x before 3.4.6
PHP 5.6 < 5.6.13, PHP 5.5 < 5.5.29 and PHP 5.4 < 5.4.45
利用过程：

  1. 不带User-Agent头，先访问一次目标主页，记下服务端返回的Cookie
  2. 生成反序列化poc（参考p神脚本）
  3. 将生成好的POC作为User-Agent，带上第一步获取的Cookie发包，第一次发包，脏数据进入Mysql数据库，第二次发包代码执行

    Joomla 3.7.0 (CVE-2017-8917) SQL注入漏洞

    影响版本 3.7.0

jupyter

Jupyter Notebook（此前被称为 IPython notebook）是一个交互式笔记本，支持运行 40 多种编程语言。

Jupyter Notebook 未授权访问漏洞

如果管理员未为Jupyter Notebook配置密码，将导致未授权访问漏洞
选择 new -> terminal 即可创建一个控制台 直接执行任意命令

kibana

Kibana 为 Elassticsearch 设计的一款开源的视图工具。

Kibana Local File Inclusion (CVE-2018-17246)

影响版本：5.6.13到6.4.3

如果能上传文件，那么就可以包含实现rce

Kibana原型链污染导致任意代码执行漏洞（CVE-2019-7609）

影响版本：5.6.15至6.6.1

libssh

libssh 是一个用以访问SSH服务的C语言开发包，通过它可以执行远程命令、文件传输，同时为远程的程序提供安全的传输通道

libssh 服务端权限认证绕过漏洞（CVE-2018-10933）

影响版本：0.6及更高版本
在libssh版本0.8.4和libssh 0.7.6 修复
利用参考p神的脚本

liferay-portal

Liferay Portal CE是一款用来快速构建网站的开源系统

Liferay Portal CE 反序列化命令执行漏洞（CVE-2020-7961）

影响版本：7.2.0 GA1及以前
利用：

  1. 准备一个恶意类 编译

放到web服务目录下

  1. 用marshalsec来生成一个适用于Jackson的POC

http://evil/是刚才启动的保存了恶意class文件的Web服务，LifExp是恶意类名

  1. 生成的Payload是Jackson使用的,按照Liferay Portal的形式，即+参数名:类名=值，来修改这个Payload

发包

log4j

Apache Log4j是一个用于Java的日志记录库，其支持启动远程日志服务器。

Apache Log4j Server 反序列化命令执行漏洞（CVE-2017-5645）

影响版本：Apache Log4j 2.8.2之前的2.x
使用ysoserial生成payload，然后直接发送给your-ip:4712端口即可
