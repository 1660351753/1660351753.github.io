JavaScript Prototype 污染攻击

js的类和函数很像，类需要用function定义

可以认为原型prototype是类Foo的一个属性,可以通过Foo.prototype来访问Foo类的原型
一个Foo类实例化出来的foo对象，可以通过foo.proto属性来访问Foo类的原型

通过赋值，可以修改一个对象的原型，这就是原型链污染
有两种操作
1.对象merge
2.对象clone（其实内核就是将待操作的对象merge到一个空对象中）
